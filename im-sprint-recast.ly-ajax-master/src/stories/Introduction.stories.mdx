# 스프린트 진행 방법

오른쪽의 컴포넌트 이름을 클릭하여, 실제로 HTML 상에서 어떻게 컴포넌트가 표시되는지 확인할 수 있습니다.

하단의 Controls라고 되어있는 부분은 props를 넘기는 부분입니다. 동적으로 props를 넘기면서 props에 따라 어떻게 컴포넌트의 모양이 바뀌는지 확인할 수 있습니다

Storybook(`npm run storybook`)을 통해 각 컴포넌트의 모양을 확인하며, `npm test`를 통해 요구사항을 통과하는지 확인할 수 있습니다.

`npm start`를 통해 실제 웹 앱이 실행되는 화면을 볼 수 있습니다.
